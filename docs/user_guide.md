# AI游戏公司 - 用户指南

> 欢迎来到AI游戏公司！这是一个由5个AI Agent协作自动开发HTML5游戏的平台。

---

## 🎮 快速开始

### 1. 启动服务器

在项目根目录下运行：

```bash
cd backend
python main.py
```

看到以下输出表示启动成功：

```
🚀 AI游戏公司服务器启动成功！
📍 地址: http://localhost:8000
📚 API文档: http://localhost:8000/api/docs
```

### 2. 打开前端页面

在浏览器中访问：**http://localhost:8000**

### 3. 创建第一个游戏

1. 点击右上角的 **▶ CREATE PROJECT** 按钮
2. 填写项目信息：
   - **项目名称**：如 `my_first_game`
   - **游戏描述**：如 `做一个简单的贪吃蛇游戏`
3. 点击 **▶ START** 开始创建

### 4. 观看开发过程

创建项目后，你会看到：

- **办公室视图**：5个AI Agent在"办公"
- **消息日志**：Agent之间的对话
- **进度条**：开发进度（0%-100%）
- **决策请求**：需要你做的决定会弹窗提示

### 5. 试玩游戏

当进度达到100%时：

1. 点击右上角的 **🎮 PLAY** 按钮
2. 游戏会在新窗口打开
3. 玩得开心！

## 🤖 5个AI Agent介绍

| Agent | 角色 | 职责 |
|-------|------|------|
| 👔 **PM** | 项目经理 | 接收需求、拆解任务、协调团队 |

| 💻 **程序** | 程序员 | 编写游戏代码(HTML/JS) |
| 🎨 **美术** | 美术设计师 | 生成游戏素材图片 |
| 🧪 **测试** | 测试工程师 | 测试游戏、报告Bug |

## 🎯 开发流程

游戏开发分为8个阶段：

```
1. 立项     → PM分析需求，组织团队
2. 策划     → 策划编写游戏设计文档
3. 技术设计 → 程序员设计技术架构
4. 并行开发 → 程序员编码 + 美术生成素材
5. 整合     → 整合代码和素材
6. 测试     → 测试工程师测试游戏
7. Bug修复  → 修复发现的问题
8. 交付     → PM汇报完成
```

每个阶段完成后会弹出**决策请求**，你可以：
- **批准继续** - 进入下一阶段
- **修改调整** - 要求重新处理
- **取消项目** - 终止开发

## ⚠️ 常见问题

### Q: 服务器启动失败怎么办？

检查以下几点：
1. 确保Python版本 ≥ 3.11
2. 确保已安装所有依赖：`pip install -r requirements.txt`
3. 确保 `.env` 文件中配置了 `GOOGLE_API_KEY`

### Q: 游戏开发卡住不动？

可能原因：
1. **网络问题**：检查网络连接
2. **API配额**：Gemini API可能有调用限制
3. **决策超时**：如果弹出决策请求，请及时选择

### Q: 生成的游戏不完整？

游戏复杂度会影响生成质量。建议：
1. 从简单游戏开始尝试
2. 在描述中明确说明游戏规则
3. 使用"首次使用引导"推荐的游戏类型

### Q: 如何重新开始？

刷新浏览器页面即可。之前的游戏文件会保存在 `projects/` 目录中。
